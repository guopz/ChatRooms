# chatrooms

一个简单的聊天室，可以支持多人聊天 Node + Socket.io

有兴趣的朋友可以 clone 学习一下，大家共同进步

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
node server 

```
